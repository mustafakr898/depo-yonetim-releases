# Depo Yönetim — Dağıtım

Bu depo yalnızca **kurulum dosyalarını** ve **sürüm bilgisini** barındırır.
Kaynak kod bu depoda **bulunmaz** (özel depoda tutulur).

- `version.json` — en güncel sürüm bilgisi (uygulama bunu okur)
- **Releases** — Windows kurulum dosyası (.exe) ve Android APK

Uygulama güncellemeleri buradan indirilir.
